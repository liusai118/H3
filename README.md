The Dataset folder is available for download at https://drive.google.com/file/d/1B_CFm9nzzaNJ7wFXYJAxSBadj3dUFNnh/view?usp=sharing.
After extraction, the folder is named Dataset. The folder contains four files:
rosetta.csv (generated by rosetta/data_analysis_04.ipynb)
neg.csv (generated by ./xai/random_02.R)
pos.csv (generated by ./xai/random_02.R)
mutant.csv (generated by ./xai/mut_04.R)
