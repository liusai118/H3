## Data Availability

### 1. Sequence Data

The **sequence** folder is available for download at:  
https://drive.google.com/file/d/14HpnidUxznzjZTv_TvWRuoV5ddk0hNdS/view?usp=sharing  

After extraction, the folder is named `sequence`. It contains the following files:

- `neg_seq.csv` (generated by `./xai/random_02.R`)
- `pos_seq.csv` (generated by `./xai/random_02.R`)
- `mut_seq.csv` (generated by `./xai/mut_06.R`)

---

### 2. Structure Data

The **structure** folder is available for download at:  
https://drive.google.com/file/d/1qZYep0-YNTFgDsANIqK04HNSdbRkrlae/view?usp=drive_link  

This folder contains the **complex structures** formed by sequences in `proteinX.csv` (predicted by ProteinX) and peptides carrying **H3K18bu**.  
These structures are used for subsequent **conformational optimization** and **screening with Rosetta**.

---

### 3. Dataset

The **Dataset** folder is available for download at:  
https://drive.google.com/file/d/1B_CFm9nzzaNJ7wFXYJAxSBadj3dUFNnh/view?usp=sharing  

After extraction, the folder is named `Dataset`. It contains the following files:

- `rosetta.csv` (generated by `rosetta/data_analysis_04.ipynb`)
- `neg.csv` (generated by `./xai/data_analysis_05.R`)
- `pos.csv` (generated by `./xai/data_analysis_05.R`)
- `mutant.csv` (generated by `./xai/data_analysis_05.R`)
